1단계 요구사항

*사다리 게임에 참여하는 이름은 최대 5글자

*사다리를 출력할 때 사람 이름도 출력

*사람 이름은 쉼표(,)를 기준으로 구분

*사람 이름 중 가장 긴 이름을 기준으로 사다리 폭 설정

*사다리 타기의 라인은 겹치지 않게 설정
